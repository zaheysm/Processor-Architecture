Question

There is a 5 byte table of unsigned integers, named T0, and a 5 word array of signed integers named A0. Write an instruction sequence using an iterative control structure which will copy the byte values of table T0 into the words of array A0.

Answer

     ldx  #T0
     ldy  #A0
     clra
l1:  ldab 1,x+     ; 16 bit value (signed/unsigned) is in D
     std  2,y+
     cpx  #T0+5
     blo  l1
