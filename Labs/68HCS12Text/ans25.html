Question

With the high reference voltage set to 4.5 volts and the low reference voltage set to 0.5 volts, and 2 volts applied to the ATD input, what measurement code is expected assuming a 10 bit, right-justified conversion?

Answer

The equation given in the text is round((V-VRL)*2bits/(VRH-VRL)). Substituting the values for V, VRL, VRH, and bits, we get, round((2-0.5)*1024/(4.5-0.5)) = round(384) = 384.
