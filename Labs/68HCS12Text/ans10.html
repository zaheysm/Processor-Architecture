Question

Write a code sequence which will increment the four byte value at location $1020 through $1023. The value is stored in big-endian order.

Answer

    ldaa $1023
    adda #1       ; increment
    staa $1023
    ldaa $1022
    adca #0       ; add any carry out of $1023
    staa $1022
    ldaa $1021
    adca #0       ; add any carry out of $1022
    staa $1021
    ldaa $1020
    adca #0       ; add any carry out of $1021
    staa $1020
