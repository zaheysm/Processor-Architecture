Question

A synchronous interface has an output propagation time (from active clock edge) of 10ns, an input setup time of 10 ns, and an input hold time of 5 ns. Assuming no clock skew (output and input devices have clock edges that occur at the same time) what is the maximum clock rate possible without breaking an input timing specification?

Answer

The output propagation time plus the input setup time must be less than or equal to the period, so the minimum period is 10+10 = 20ns. This gives a maximum clock rate of 50 MHz.
