Question

What sequence of subroutine calls using the supplied subroutine library will write the sequence of bytes $10, $11, to bus address $30?

Answer

Address $30 is binary 0110000, to which we append the 0 bit for a write operation. That gives 01100000.
Call IICSTART with argument %01100000
call IICTRANSMIT with argument $10
call IICTRANSMIT with argument $11
call IICSTOP
