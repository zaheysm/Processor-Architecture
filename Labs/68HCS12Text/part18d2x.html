In this program you will need to use the signal generator (available from the view menu) to create a waveform on Port T pin 7. One measurement will be made each second of simulation time, and the frequency will be stored in the 4 byte variable starting at location $1000.

The Simulator will appear in a separate window. When this page is exited, the simulator will stop. The controls in the simulator for loading code, generating a log file, and taking a snapshot are disabled when run from the browser.
