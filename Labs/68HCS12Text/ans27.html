Question

How does one access physical address $E5002?

Answer

The upper six bits of the physical address is the page, and the lower 14 bits are the offset into the page. In this problem, with the binary address 1110 0101 0000 0000 0010 (Spaces added for clarity), the page is 111001, or $39, and the offset is 01 0000 0000 0010, or $1002. Since this can only be visible in the window starting at $8000, we need to set PPAGE=$39 and reference location $9002 ($8000+$1002).
