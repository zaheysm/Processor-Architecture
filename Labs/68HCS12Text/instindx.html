68HC12 CPU Instruction Index

This page has links to where each instruction is discussed in the text. If you are not sure what the instruction mnemonic you are interested in is, check the S12CPUV2 Reference Manual. Similar instructions are grouped together, based on the first mnemonic in alphabetical order. Use your browser search function to find the mnemonic. Instruction mnemonics in italics are aliases which are provided for upward compatibility from the earlier 68HC11 processor.
