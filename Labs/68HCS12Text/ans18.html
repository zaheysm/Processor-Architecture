Question

Port M pin 3 is to be used as an input. There is to be no internal pullup/pulldown used. What sequence of instructions will configure port M to insure this operation? What instruction can be used to test and branch to location foo on port M pin 3 being logical '1'?

Answer

To configure:

    bclr ddrm #$8
    bclr perm #$8

To test and branch:

    brset ptim #$8 foo
