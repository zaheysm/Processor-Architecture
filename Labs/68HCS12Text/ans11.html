Question

Write a code sequence using an iteration control structure which will clear memory locations $2000 through $200F.

Answer

    ldx  #$2000
l1: clr  1,x+
    cpx  #$200F
    bls  l1        ; iterate while address <= $200F

This could also be done using movb #0 1,x+ which would be slower, or even clearing an accumulator and then storing the value with, for instance, staa 1,x+. An alternative which doesn't use the auto-increment indexed addressing mode:

    ldx  #$2000
l1: clr  0,x
    inx
    cpx  #$200F
    bls  l1        ; iterate while address <= $200F

Another alternative which uses a counter:

    ldaa #16       ; number of bytes
    ldx  #$2000    ; starting address
l1: clr  1,x+
    dbne a l1      ; repeat until a decrements to zero
