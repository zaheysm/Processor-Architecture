HCS12 Documentation

All of these documents are available from the Freescale Semiconductor website and are provided here for your convenience. Please go to the Freescale Semiconductor site to download the latest revisions.

These documents apply to the MC9S12DP256 as well as the DT, DJ, DG, and A variations.

  • MC9S12DP256 Device Users Guide (overview, has Electrical characteristics, Register Map)
  • ATD_10B8C Block Users Guide (Analog to Digital Converter)
  • HCS12 V1.5 Core Users Guide (CPU Core, BDM interface)
  • S12CPUV2 Reference Manual (CPU Instructions, this is the programmers' manual)
  • CRG Block Users Guide (Clock and Reset Generator)
  • MC9S12DP256 Port Integration Module Block Users Guide (External interface for ports A,B,E,H,J,K,M,P,S,T)
  • ECT_16B8C Block Users Guide (Enhanced Timer Capture)
  • EETS4K Block Users Guide (4K EEPROM)
  • FTS256K Block Users Guide (256K Flash EEPROM)
  • PWM_8B8C Block Users Guide (Pulse Width Modulator)
  • HCS12 Serial Communications Interface Block Guide
  • SPI Block Users Guide (Serial Peripheral Interface)
  • HSC12 Inter-Integrated Circuit Block Guide
  • Reference Guide for D-Bug12
  • BDLC Block Guide
  • MSCAN Block Guide (CAN interface)
